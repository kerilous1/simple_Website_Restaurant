# 🍔 Restaurant Management System

A desktop application designed to streamline restaurant operations, including user login, order management, and interactive GUI.

## ✨ Features
* **User Authentication:** Secure login system for admins and staff.
* **Interactive Dashboard:** Easy-to-use graphical interface.
* **Order Tracking:** Add, edit, and manage customer orders efficiently.
